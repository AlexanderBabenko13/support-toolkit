\# Release Notes



\## v0.8



Релизная подготовка SupportToolkit.



\### Добавлено



\- Обновлена версия тулзы до 0.8.

\- Добавлена portable-сборка через build/Build-SingleFile.ps1.

\- Добавлена сборка архива dist/SupportToolkit\_v\*.zip.

\- Добавлена инструкция запуска portable-версии.

\- dist/ исключён из Git через .gitignore.



\### Уже есть в тулзе



\- Диагностика системы.

\- Проверка дисков.

\- Проверка критичных служб.

\- Последние ошибки Event Log.

\- Сетевая диагностика.

\- DNS-информация.

\- Проверка DNS-имени.

\- Проверка адреса или порта.

\- Проверка адресов из локального конфига.

\- Быстрый отчёт.

\- Экспорт TXT.

\- Экспорт HTML.

\- Локальный конфиг endpoint-ов без публикации в GitHub.



\### Принципы безопасности



\- По умолчанию только read-only диагностика.

\- Системные настройки не меняются.

\- Службы не перезапускаются.

\- DNS-кэш не очищается.

\- Локальные endpoint-ы не публикуются в GitHub.
