\# SupportToolkit



PowerShell-утилита для L2 поддержки.



\## Цель проекта



Собрать безопасный набор диагностических инструментов для инженера поддержки:



\- информация о системе;

\- информация о сети;

\- проверка адресов и портов;

\- экспорт диагностического отчета;

\- дальнейшее расширение под VPN, DNS, RDP, Outlook, printers и AD.



\## Принципы



1\. Скрипт не должен менять систему без подтверждения.

2\. По умолчанию только диагностика.

3\. Все действия должны логироваться.

4\. Код должен быть понятным и разбитым на функции.

5\. Без сторонних модулей, если это возможно.

6\. Совместимость с PowerShell 7.

7\. Вывод на русском языке.
